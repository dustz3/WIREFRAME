# TailorMed 線框圖預覽指南

## 快速開始

1. **編譯專案**
   ```bash
   npm run build
   ```

2. **開啟預覽**
   - 在瀏覽器中開啟 `dist/index.html`
   - 或使用 Live Server 擴展進行即時預覽

## 線框圖特色

### 🎨 設計風格
- **清潔臨床**：採用 Biocair 風格的簡潔設計
- **專業色彩**：藍灰色調 (#2563EB) 搭配大量留白
- **現代字體**：Inter 字體確保最佳可讀性
- **響應式**：適配桌面、平板和手機

### 📱 響應式設計
- **桌面版**：完整功能展示，多欄佈局
- **平板版**：適配中等螢幕，調整間距
- **手機版**：單欄佈局，優化觸控體驗

### 🔧 技術特色
- **Pug 模板**：語法簡潔，易於維護
- **Stylus 預處理器**：變數和混入，提高開發效率
- **SVG 圖標**：內嵌圖標，載入快速
- **語義化 HTML**：良好的 SEO 和可訪問性

## 開發工作流程

### 修改模板
1. 編輯 `src/templates/index.pug`
2. 執行 `npm run build`
3. 重新整理瀏覽器

### 修改樣式
1. 編輯 `src/styles/main.styl`
2. 執行 `npm run build`
3. 重新整理瀏覽器

### 監聽模式
```bash
npm run dev
```
這會自動監聽檔案變化並重新編譯。

## 瀏覽器測試

建議在以下瀏覽器中測試：
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
